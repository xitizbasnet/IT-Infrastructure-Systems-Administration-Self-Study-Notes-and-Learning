# ­¤¦æŌĆŹ­¤Æ╗ Step 2: Windows ŌĆö via PowerShell

## Configure Static IP Address

```powershell id="winpsip01"

```

---

## Configure DNS Servers

```powershell id="windns01"
Set-DnsClientServerAddress -InterfaceAlias 'Ethernet' -ServerAddresses 192.168.1.10,192.168.1.11
```

---

## Parameter Reference

| Parameter         | Description               |
| ----------------- | ------------------------- |
| `InterfaceAlias`  | Network adapter name      |
| `IPAddress`       | Static IP address         |
| `PrefixLength`    | Subnet mask (CIDR format) |
| `DefaultGateway`  | Router IP                 |
| `ServerAddresses` | DNS servers               |

# ­¤É¦ Step 3: Linux ŌĆö via nmcli

## Configure Static IP Address

```bash id="linuxip01"
nmcli con mod eth0 ipv4.addresses 192.168.1.50/24
nmcli con mod eth0 ipv4.gateway 192.168.1.1
nmcli con mod eth0 ipv4.dns "192.168.1.10 8.8.8.8"
nmcli con mod eth0 ipv4.method manual
nmcli con up eth0
```

---

## Command Breakdown

| Command              | Purpose                     |
| -------------------- | --------------------------- |
| `ipv4.addresses`     | Assign IP and subnet        |
| `ipv4.gateway`       | Set default route           |
| `ipv4.dns`           | Configure DNS servers       |
| `ipv4.method manual` | Enable static configuration |
| `con up`             | Activate connection         |


| Parameter     | Description         |
